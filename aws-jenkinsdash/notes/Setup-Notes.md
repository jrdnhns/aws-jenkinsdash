## Objective
Deploy Jenkins on AWS EC2 Ubuntu instance.
## Issues Encountered
- Jenkins repository version mismatch
- SSH key path troubleshooting
- Java compatibility issue prevented Jenkins startup
## Resolution Steps
- Corrected Jenkins repository source
- Verified PowerShell SSH key location
- Installed supported OpenJDK version
- Restarted Jenkins service
## Technologies Used
- AWS EC2
- Ubuntu Linux
- PowerShell
- SSH
- Jenkins
- OpenJDK
- GitHub
