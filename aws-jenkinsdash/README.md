	# AWS Jenkins Lab
## Overview
This project demonstrates deploying and configuring Jenkins on an Ubuntu EC2 instance hosted in AWS.
## Technologies Used
- AWS EC2
- Ubuntu Server
- Jenkins
- OpenJDK 21
- PowerShell
- SSH
- GitHub
## Skills Demonstrated
- Cloud deployment
- Linux administration
- SSH remote access
- Jenkins configuration
- Service troubleshooting
- Security group configuration
## Key Troubleshooting
A major issue encountered during deployment was a Jenkins startup failure caused by Java compatibility problems. Troubleshooting involved verifying Java versions, reinstalling a supported OpenJDK release, and restarting Jenkins services successfully.
